Alguns Serviços realizados

Lótus - SPA & Terapia

O primeiro site a gente nunca esqueçe...

Este foi um dos primeiros sites que desenvolvi na época dos cursos técnicos, utilizando somente HTML e CSS. A Lótus foi uma empresa fictícia desenvolvida para testar os conceitos básicos de webdesign e design gráfico.

Tecnologias Utilizadas: HTML, CSS, JS

Astrocalc

Uma calculadora para entusiastas da Astronomia

Este projeto foi desenvolvido para ajudar a estudantes e entusiastas de Astronomia a realizar cálculos astronômicos simples, como o Raio de Schwarzschild Radius, as Leis de Kepler, Wein e da Gravitação Universal Newtoniana.

Tecnologias Utilizadas: HTML, CSS, JS, GIT/Github

SimcirJS

Simulador de Circuitos Digitais para Web

O SimcirJs é um simulador de circutos digitais open-source criado por Kazuhiko Arase em 2014. Realizei algumas implementações no aplicativo, criando um logotipo para o projeto e desenvolvendo alguns componentes (LEDs verde, azul e amarelo) para melhor ilustrar os conceitos de Eletrônica Digital nas aulas de programação e manutenção que ministrei em algumas escolas.

Tecnologias Utilizadas: HTML, CSS, JS, GIT/Github, SVG

Arduino Signals

Um osciloscópio web para Arduino

Projeto desenvolvido para auxiliar em um Trabalho de Iniciação Científica desenvolvido por alguns colegas e eu em 2016/2017.

Tecnologias Utilizadas: HTML, CSS, JS, GIT/Github, PHP, Arduino
